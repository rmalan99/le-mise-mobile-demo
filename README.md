# le mise

`le-mise` is a digital recipe ecosystem focused on home cooking, designed to help people discover what to cook more effectively, make decisions faster based on their context, and execute the full cooking process with less friction.

## Purpose

The product is built to solve three concrete problems:

- finding relevant recipes based on real user needs,
- making decisions easier through filters, categories, and usage context,
- connecting the experience between discovery, recipe detail, and guided cooking.

## Value Proposition

`le-mise` aims to bring together in a single experience:

- recipe exploration and search,
- culinary content organization,
- filters based on preferences and needs,
- a clear recipe detail experience,
- continuity between choosing, preparing, and cooking.

## Current Status

In this demo, the focus is on the mobile app, which acts as the main end-user consumption surface.

The current implementation represents an initial version of the product with:

- mobile navigation,
- a local catalog,
- recipe discovery,
- recipe detail reading,

as a foundation to evolve into a more complete solution.

## Tech Stack

This demo is currently built with:

- React
- Ionic
- Tailwind CSS
- Vite

## Disclaimer

This application is a basic demo of a personal project.

Its goal is not to remain an isolated experiment, but to serve as the first visible version of a product intended to evolve into a real, usable solution with value for real users.
